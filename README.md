MAD204-LAB04-JennyferParmar
This project  is an Android Notes application developed using Kotlin as part of the MAD204 Lab 4 assignment.  
The app allows users to create, view, edit, and delete notes stored locally using a Room database.  
It also demonstrates the use of core Android components such as Activities, RecyclerView, Services, coroutines, and notifications.

Features
- Add notes with a title and content  
- View all saved notes in a RecyclerView  
- Edit notes using a dialog interface  
- Delete notes with an Undo option using Snackbar  
- Store data locally using Room database  
- Perform database operations using Kotlin coroutines  
- Start a background Service that displays a reminder notification  

Technologies Used
- Kotlin  
- Android SDK  
- Room (Entity, DAO, Database)  
- RecyclerView  
- Android Services  
- Kotlin Coroutines  
- Material Design Components  

Project Structure
- MainActivity.kt – Handles UI logic and user interaction  
- Note.kt – Room entity representing a note  
- NoteDao.kt – Data Access Object for database operations  
- NotesDatabase.kt – Room database configuration  
- NoteAdapter.kt – RecyclerView adapter for displaying notes  
- ReminderService.kt – Background service for reminders  
- activity_main.xml – Main UI layout  
- dialog_edit_note.xml – Layout for editing notes  

How to Run the Project
1. Clone or download the repository  
2. Open the project in Android Studio  
3. Let Gradle sync complete  
4. Run the app on an emulator or physical Android device
